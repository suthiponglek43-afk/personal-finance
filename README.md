# 💰 Expense Tracker PWA

แอปบันทึกรายจ่าย Progressive Web App — ใช้งานออฟไลน์ได้ ติดตั้งบนมือถือเหมือน Native App

## Features

- 📊 Dashboard สรุปรายจ่ายพร้อมกราฟ
- 👛 กระเป๋าตังค์ — เงินสด, ธนาคารไทย 14 แห่ง (แยกบัญชีย่อย), พอร์ตลงทุน
- 📈 Analytics — เปรียบเทียบสัปดาห์/เดือน/ปี
- 💰 งบประมาณเดือนต่อเดือน
- 🌐 ไทย / English
- 📴 ใช้งานออฟไลน์ได้
- 📲 ติดตั้งบนหน้าจอหลักได้
- 📤 Export CSV

## Quick Start

```bash
npm install
npm run dev
```

## Deploy to Vercel

1. Push repo นี้ขึ้น GitHub
2. ไปที่ [vercel.com](https://vercel.com) → New Project → Import repo → Deploy
3. เปิด URL จากมือถือ → Add to Home Screen

## Tech Stack

React 18 · Vite 5 · Recharts · IndexedDB · PWA (Workbox)
